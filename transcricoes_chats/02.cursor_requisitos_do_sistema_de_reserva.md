## Estrutura do Projeto

O projeto está em `/home/leonardo/Downloads/AI4SE_cursor/hotel-reservation` e segue a arquitetura modular definida nas regras do projeto.

### Módulos Implementados

**1. Módulo de Quartos (`/src/modules/rooms/`)**
- Cadastro com: número, capacidade, tipo (Básico/Moderno/Luxo), preço por diária
- Comodidades: frigobar, café da manhã, ar-condicionado, TV
- Seção de camas com tipos: Solteiro, Casal King, Casal Queen
- Listagem com disponibilidade editável (Ocupado/Livre/Manutenção/Limpeza)

**2. Módulo de Hóspedes (`/src/modules/guests/`)**
- Cadastro com: nome, sobrenome, CPF (com validação), e-mail
- Listagem exibe: nome, sobrenome, CPF (formatado)
- Validação de CPF com dígitos verificadores

**3. Módulo de Reservas (`/src/modules/reservations/`)**
- Seleção de quarto disponível e hóspede cadastrado
- Datas de check-in/check-out com validação
- Status: Ativa, Concluída, Cancelada
- Listagem com chips coloridos para disponibilidade

### Tecnologias Utilizadas

- **React 18** + **TypeScript** (strict mode)
- **Vite 5** como build tool
- **Material-UI** com tema verde/azul
- **React Query** para gerenciamento de estado
- **React Hook Form** + **Zod** para formulários e validação
- **LocalStorage** para persistência de dados

### Como Executar

Para iniciar novamente:
```bash
cd hotel-reservation
npm run dev
```

Para build de produção:
```bash
npm run build
```
